
# Lab 5 — SaaS Tasker Landing Page

## Використані технології та інструменти

- **Vue 3 (Composition API)**
- **PrimeVue** — бібліотека UI-компонентів.
- **Aura** — тема PrimeVue, використана для стилізації.
- **PrimeIcons** — набір іконок.
- **Vite** — інструмент для збірки проєкту.
- **CopyToClipboardPlugin** — кастомний плагін для копіювання тексту в буфер обміну.

## Структура та реалізовані секції

Лендінг реалізовано як односторінковий додаток, що складається з 10 компонентів-секцій:

1.  **Навігація:** `SiteHeader` (з якорем-скролом на секції).
2.  **Основний контент:** `HeroSection`, `ProblemSolution`, `AdvantagesSection`, `HowItWorks`.
3.  **Бізнес-логіка:** `PricingPlans` (з калькулятором), `ReviewsSection`, `FAQSection`.
4.  **Комунікація:** `ContactForm` (з валідацією та `Toast` повідомленням).
5.  **Фінальний блок:** `SiteFooter` (з функцією копіювання Email).

##  Функціональність та використання компонентів

### PrimeVue

Компоненти PrimeVue (включно з `Card`, `Button`, `Toast`, `InputText`, `InputMask`, `InputNumber`, `Dropdown` ) використані для всіх інтерактивних та контентних елементів.

**Ключова особливість:** У всіх випадках для компонента `Card` коректно використані **іменовані слоти** (`#header`, `#title`, `#content`, `#footer`) для забезпечення правильного рендерингу вмісту.

### Функціональність

* **Навігація:** Реалізовано плавний скрол (scroll-to-section) до цільових секцій за допомогою програмного методу `element.scrollIntoView({ behavior: 'smooth' })`.
* **Контактна форма:**
    * Реалізовано просту клієнтську валідацію (`validate()` функція).
    * Використовується `ToastService` для відображення повідомлення про успішне надсилання.
* **Калькулятор:** У секції `PricingPlans` реалізовано інтерактивний калькулятор вартості з використанням `InputNumber` та `Dropdown`.
* **Копіювання Email:** У `SiteFooter` реалізовано функцію копіювання Email-адреси в буфер обміну через підключений плагін `CopyToClipboardPlugin`.

## Налаштування

PrimeVue налаштовано у `main.js`:

* **Тема:** `Aura` (preset)
* **Сервіси:** `ToastService`
* **Стилі:** `primeicons.css`
