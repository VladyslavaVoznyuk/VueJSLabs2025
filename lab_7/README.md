# lab_7

### Тести
Unit / Component Tests (Vitest + Vue Test Utils)

Запуск тестів:
```sh
npm run test:unit
```

Графічний UI-режим:

```sh
npm run test:unit:ui
```
### Coverage (Покриття тестами)

Команда для генерації покриття:
```sh
npm run test:unit -- --coverage
```

### Playwright

Запуск усіх E2E:
```sh
npm run test:e2e
```
UI для налагодження тестів:
```sh
npm run test:e2e:ui
```

| Тип тестів | Покриття                           | Опис                                 |
| ---------- | ---------------------------------- | ------------------------------------ |
| Unit tests | ~90–100%  | Протестована логіка компонентів Todo |
| E2E tests  | Перевіряють ключові сценарії UI    | Перевірка поведінки в браузері       |

### Що покрито:

Компонент App.vue:

додавання задачі

відображення списку

видалення задачі

зміна статусу “Completed”

### Що не покрито:

Валідація введення

UX-поведінка у порожньому списку

### Реалізовані тести
 Unit Tests:

Перевірка рендерингу початкових tasks

Перевірка додавання нової задачі

Перевірка видалення задачі

Перевірка зміни статусу чекбоксом

### E2E сценарії (Playwright):

 Відкриття сторінки Todo

 Додавання задачі через UI

 Перемикання статусу виконання

 Видалення задачі
